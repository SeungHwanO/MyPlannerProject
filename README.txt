[git terminal guide]

git init : 로컬 깃 파일 초기화
git config --global user.email "" : 버전 관리를 위한 email 정보 등록
git config --global user.name "" : 버전 관리를 위한 name 정보 등록
git add . : 커밋에 추가할 파일 선택 ( . 모든파일)
git commit -m "" : 커밋 상세 설명

git log : 커밋 기록 확인
git checkout 커밋주소 : 해당 커밋으로 체크아웃

remote add origin 깃허브주소 : 원격저장소 주소 지정
git push origin 브랜치이름 : 로컬저장소 커밋들을 해당 브랜치로 업로드
